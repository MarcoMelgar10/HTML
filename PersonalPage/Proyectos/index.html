Biblioteca

Descripción del Proyecto

Sistema integral para la administración de una biblioteca, que permite gestionar usuarios, libros, préstamos y devoluciones...

Gestión de usuarios: registrar y consultar lectores.
Administración de libros: catálogo detallado y búsqueda avanzada.
Préstamos y devoluciones: control de ejemplares y vencimientos.
Historial y reportes: estadísticas de uso y libros más solicitados.
Interfaz intuitiva: navegación clara y accesible.

Tecnologías Utilizadas

  • Java con Swing para interfaz.
  • PostgreSQL como base de datos.
  • JDBC para la conexión.
  • API WhatsApp para notificaciones.
  • Ubuntu Server como entorno de despliegue.

DogDer

Descripción del Proyecto

Aplicación móvil inspirada en Tinder, diseñada para conectar a dueños de perros que buscan socializar o encontrar compañeros para sus mascotas...

Perfiles: creación de tarjeta para cada perro.
Exploración: swipes para descubrir otros perros.
Match: coincidencias habilitan chat.
Filtros: búsqueda por raza, edad, género o ubicación.

Tecnologías Utilizadas

  • Kotlin para Android.
  • Jetpack Components (ViewModel, LiveData, Room, Navigation).
  • RecyclerView + CardStackView para swipes.
  • Firebase (Firestore, Authentication, Storage, Messaging).
  • Glide / Coil para carga de imágenes.

Pintore

Descripción del Proyecto

Aplicación de escritorio en Java para edición básica e interactiva de imágenes...

Herramientas de dibujo: pinceles, cuadrados, líneas (Bresenham).
Carga de imágenes: apertura desde sistema de archivos.
Colores: JColorChooser y toolbar intuitiva.
Historial: patrón Command para deshacer.
Múltiples imágenes: pestañas con historiales independientes.

Tecnologías Utilizadas

  • Java con Swing.
  • PostgreSQL como base de datos.
  • JDBC para conexión.
  • Patrones de diseño: Command, MVC, Observer.
